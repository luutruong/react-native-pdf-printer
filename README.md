# react-native-pdf-printer

Print local PDF file

## Installation

```sh
npm install react-native-pdf-printer
```

## Usage

```js
import { printPdf } from 'react-native-pdf-printer';

// ...

const result = await printPdf("/path/to/local/pdf/file.pdf");
```
